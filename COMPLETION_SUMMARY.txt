💻 USAGE AFTER SETUP:
════════════════════════════════════════════════════════════════════════════════

In your game code, simply use:

  // Open search menu
  window.menuSystem.openSearch((levelId) => {
    console.log('Loading level:', levelId);
  });

  // Open level select
  window.menuSystem.openLevelSelect([levels], (level) => {
    console.log('Playing:', level);
  });

  // Open featured
  window.menuSystem.openFeatured([levels], (level) => {
    console.log('Featured:', level);
  });

  // Close any menu
  window.menuSystem.hide();
