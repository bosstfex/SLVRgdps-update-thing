💻 USAGE AFTER SETUP:
════════════════════════════════════════════════════════════════════════════════

In your game code, simply call:

// Open search menu
window.menuSystem.openSearch((levelId) => {
  console.log('Loading level:', levelId);
});

// Open level select
window.menuSystem.openLevelSelect([
  { id: '1', name: 'Level 1' }
], (level) => {
  console.log('Selected:', level);
});

// Open featured
window.menuSystem.openFeatured([
  { name: 'Featured', description: 'Cool level' }
], (level) => {
  console.log('Playing:', level);
});

// Close any menu
window.menuSystem.hide();
